---
name: forge-bootstrap
description: Initialize .claude/ in a new project (--profile minimal|standard|full)
---

Run `/forge bootstrap`. Use `$DOTFORGE_DIR/template/` as base. Auto-detect project stacks and layer matching configs. Profile argument via $ARGUMENTS (default: standard). Generates CLAUDE.md, settings.json, rules, hooks, commands, agents, and CLAUDE_ERRORS.md.
