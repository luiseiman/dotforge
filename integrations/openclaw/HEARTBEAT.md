# Heartbeat checklist

> Checks every 30 minutes during active hours (9 AM - 11 PM ART)

- [ ] Scan Telegram channels for unread messages requiring action
- [ ] Check Gmail for urgent or time-sensitive emails
- [ ] Review calendar — anything in the next 2 hours that needs prep?

> Every 2 hours

- [ ] Check GitHub notifications — open PRs, review requests, issue mentions
- [ ] Any blocked tasks from previous sessions? Note what's missing.

> Daily (run once, morning)

- [ ] Pull latest on monitored repos (SOMA, Cotiza-Api-Cloud, InviSight, claude-kit)
- [ ] Check if any Supabase migrations are pending
- [ ] Review Obsidian vault for unprocessed decisions or postmortems

> Weekly (Monday morning)

- [ ] Run `/forge audit` on registered projects — flag score drops
- [ ] Check claude-kit practices inbox for unprocessed items
- [ ] Review trading bot performance logs if available
